# Opportunity: AI Agent Setup Service

## The bet
Small businesses and solo founders pay us to build and deploy their own Claude-powered agent stack — because they want AI automation but can't build it themselves.

## Target customer
- Solo founders, indie hackers, small agency owners (1-10 people)
- Pain: they know AI can run parts of their business but have no idea how to set it up
- They find us via: Twitter/X, Indie Hackers, ProductHunt, word of mouth

## Revenue model
- Fixed-price setup packages: **$500 (starter)** / **$1,500 (full stack)**
  - Starter: 3 agents (ops, dev, reporting) configured for their business
  - Full stack: full roster customized to their workflow + 30-day async support
- Optional: **$200/mo retainer** for ongoing agent maintenance and new agent builds
- First dollar possible in under 1 week if we land one client

## Go-to-market
1. Post a detailed breakdown of this company's agent setup on Twitter/X and Indie Hackers
2. Offer the first 3 setups at 50% off in exchange for testimonials
3. DM 10 solo founders who have publicly complained about ops/admin overhead
4. Create a simple order form (Gumroad or Stripe link) — no website needed yet

## Risks
- Clients may expect ongoing support we can't scale without more agents
- Highly custom work is hard to productize — scope creep risk on fixed-price deals
- Claude API costs eat margin if clients have high agent run volume
