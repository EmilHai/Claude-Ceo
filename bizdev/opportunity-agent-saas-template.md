# Opportunity: Sell the Agent Stack as a Template / Repo

## The bet
Developers and technical founders pay a one-time fee to get the full agent stack as a ready-to-run repo they can drop into their own projects.

## Target customer
- Technical founders and indie devs who use Claude Code or build with the Anthropic API
- Pain: they want an autonomous agent loop for their company but don't want to architect it from scratch
- They find us via: GitHub, Hacker News, Claude/Anthropic Discord, Twitter/X

## Revenue model
- One-time digital product: **$29–$79** via Gumroad or Lemon Squeezy
  - Includes: all agent .md files, CLAUDE.md, folder structure, example tasks
  - Optional: **$149** tier includes a 30-min onboarding call
- Volume play — low touch, high margin, sell to many
- First dollar possible in under 3 days if we post to the right places

## Go-to-market
1. Polish the repo and push to GitHub (public or private with paywall)
2. Post on Hacker News "Show HN" — frame it as "I built a company that runs on Claude agents"
3. Submit to ProductHunt
4. Cross-post on Indie Hackers and relevant subreddits (r/ClaudeAI, r/SideProject)

## Risks
- Easy to copy once published — limited moat without ongoing updates
- Anthropic could ship something similar natively, commoditizing the template
- Requires polished docs and onboarding or buyers will ask for refunds
